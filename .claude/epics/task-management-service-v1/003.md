# Task: Implement Time and Date Display Logic

## Description

Implement the `update_time()` method that retrieves the current system time and date, formats them appropriately, and updates the QLabel widgets. This task focuses on the datetime logic without the automatic timer (timer integration comes in Task 004).

## Acceptance Criteria

- [ ] `update_time()` method added to ClockWindow class
- [ ] Method retrieves current time using `datetime.now()`
- [ ] Time formatted as HH:MM:SS using strftime
- [ ] Date formatted as YYYY年MM月DD日 [Day of Week] using strftime
- [ ] `time_label` updated with formatted time string
- [ ] `date_label` updated with formatted date string
- [ ] Initial call to `update_time()` in `__init__()` displays current time
- [ ] Time and date display correctly when application launches

## Technical Details

### Implementation Approach

Add the following import and method to `clock_app.py`:

```python
from datetime import datetime

# Inside ClockWindow class:
def update_time(self):
    """Update the time and date labels with current system time"""
    now = datetime.now()

    # Format time as HH:MM:SS
    time_str = now.strftime("%H:%M:%S")
    self.time_label.setText(time_str)

    # Format date as YYYY年MM月DD日 Day-of-Week
    date_str = now.strftime("%Y年%m月%d日 %A")
    self.date_label.setText(date_str)
```

In `__init__()`, add call to initial update:
```python
# At end of __init__(), after layout setup
self.update_time()  # Initial display
```

### Key Considerations

1. **Time Format**:
   - Use `%H:%M:%S` for 24-hour format
   - Example: "16:52:01"

2. **Date Format**:
   - Use `%Y年%m月%d日 %A` for Chinese-style date with English day
   - Example: "2025年11月26日 Tuesday"
   - `%A` gives full day name in system locale

3. **datetime.now()**:
   - Always fetches current system time
   - Prevents cumulative drift
   - No timezone handling needed (uses local time)

4. **Initial Call**:
   - Call `update_time()` once in `__init__()` to show time immediately
   - Without timer, time won't update automatically yet
