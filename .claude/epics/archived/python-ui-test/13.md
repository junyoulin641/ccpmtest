**ChartBuilder Implementation (core/chart_builder.py):**

Static class for creating matplotlib figures:
```python
class ChartBuilder:
    @staticmethod
    def create_line_chart(df: pd.DataFrame, x_col: str, y_col: str, **kwargs):
        fig, ax = plt.subplots(figsize=(10, 6))
        ax.plot(df[x_col], df[y_col], **kwargs)
        ax.set_xlabel(x_col)
        ax.set_ylabel(y_col)
        ax.set_title(kwargs.get('title', f'{y_col} vs {x_col}'))
        ax.grid(True, alpha=0.3)
        return fig

    @staticmethod
    def create_bar_chart(df: pd.DataFrame, x_col: str, y_col: str, **kwargs):
        fig, ax = plt.subplots(figsize=(10, 6))
        ax.bar(df[x_col], df[y_col], **kwargs)
        ax.set_xlabel(x_col)
        ax.set_ylabel(y_col)
        ax.set_title(kwargs.get('title', f'{y_col} by {x_col}'))
        plt.xticks(rotation=45, ha='right')
        return fig

    @staticmethod
    def create_scatter_chart(df: pd.DataFrame, x_col: str, y_col: str, **kwargs):
        fig, ax = plt.subplots(figsize=(10, 6))
        ax.scatter(df[x_col], df[y_col], **kwargs)
        ax.set_xlabel(x_col)
        ax.set_ylabel(y_col)
        ax.set_title(kwargs.get('title', f'{y_col} vs {x_col}'))
        ax.grid(True, alpha=0.3)
        return fig

    @staticmethod
    def create_pie_chart(df: pd.DataFrame, labels_col: str, values_col: str, **kwargs):
        fig, ax = plt.subplots(figsize=(8, 8))
        ax.pie(df[values_col], labels=df[labels_col], autopct='%1.1f%%', **kwargs)
        ax.set_title(kwargs.get('title', f'Distribution of {values_col}'))
        return fig
```

**Matplotlib Integration:**

```python
from matplotlib.backends.backend_qt5agg import FigureCanvasQTAgg as FigureCanvas
from matplotlib.backends.backend_qt5agg import NavigationToolbar2QT as NavigationToolbar
from matplotlib.figure import Figure

class VizScreen(QWidget):
    def __init__(self):
        super().__init__()
        self.figure = Figure(figsize=(10, 6))
        self.canvas = FigureCanvas(self.figure)
        self.toolbar = NavigationToolbar(self.canvas, self)

        layout = QVBoxLayout()
        layout.addWidget(self.toolbar)
        layout.addWidget(self.canvas)
        self.setLayout(layout)
```

**Chart Generation Flow:**

1. User selects chart type from QComboBox
2. Column selectors populate with available columns from DataManager
3. User selects X and Y columns
4. User clicks "Generate Chart"
5. Validate selections (columns exist, data compatible)
6. Call ChartBuilder.create_XXX_chart()
7. Clear previous figure: `self.figure.clear()`
8. Add chart to figure: `self.canvas.figure = fig`
9. Refresh canvas: `self.canvas.draw()`

**Column Selector Logic:**

```python
def update_column_selectors(self):
    df = self.data_manager.get_data()
    if df is not None:
        columns = df.columns.tolist()
        self.x_combo.clear()
        self.y_combo.clear()
        self.x_combo.addItems(columns)
        self.y_combo.addItems(columns)
        # Set default selections
        if len(columns) > 1:
            self.y_combo.setCurrentIndex(1)
```

**Chart Type Handling:**

- Line: Requires numeric Y, any X
- Bar: Categorical X, numeric Y (limit to <100 categories)
- Scatter: Numeric X and Y
- Pie: Categorical labels, numeric values (limit to <20 slices)

**Export to PNG:**

```python
def export_chart(self):
    file_path, _ = QFileDialog.getSaveFileName(
        self,
        "Export Chart",
        "",
        "PNG Image (*.png);;PDF Document (*.pdf)"
    )
    if file_path:
        self.figure.savefig(file_path, dpi=300, bbox_inches='tight')
        QMessageBox.information(self, "Success", f"Chart saved to {file_path}")
```

**Performance Considerations:**

- Limit data points for line/scatter (downsample if >10k points)
- Aggregate data for bar charts with many categories
- Cache figures to avoid regeneration
- Use `plt.close()` to free memory after canvas update

**Error Handling:**

- No data loaded: Disable chart generation
- Invalid column selection: Show warning
- Data type mismatch: Suggest alternative chart
- Empty data after filtering: Show message

**Key Considerations:**
- Keep matplotlib and Qt event loops separate
- Use tight_layout() to prevent label cutoff
- Clear figure before each new chart to prevent memory leaks
- Provide meaningful error messages for invalid selections
