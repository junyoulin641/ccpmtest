## Technical Details

**DataManager Implementation (core/data_manager.py):**

Singleton pattern to ensure one instance:
```python
class DataManager(QObject):
    # Signals
    dataLoaded = pyqtSignal(int, int)  # rows, columns
    dataChanged = pyqtSignal()
    dataCleared = pyqtSignal()

    _instance = None

    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance

    def __init__(self):
        if not hasattr(self, '_initialized'):
            super().__init__()
            self._data = None  # pandas DataFrame
            self._file_path = None
            self._initialized = True
```

- `load_data(df: pd.DataFrame, file_path: str)` - Set current DataFrame
