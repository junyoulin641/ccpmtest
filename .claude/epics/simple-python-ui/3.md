# Task: Build tkinter UI framework with all widgets

## Description

Implement the complete tkinter UI framework in `main.py` with all required widgets: window, title label, input field, execute button, result display area, and status bar. Set up the layout using the pack() geometry manager and ensure proper visual hierarchy.

## Acceptance Criteria

- [ ] Root window created with title "Simple Python UI Tool"
- [ ] Window size set to 800x600 and is resizable
- [ ] Title label displays at the top with larger font
- [ ] Single-line text input field (Entry widget) is present and full-width
- [ ] Execute button is centered and prominent
- [ ] Result display area (Text widget, read-only) is present and expandable
- [ ] Status bar (Label) is present at the bottom showing "Idle"
- [ ] All widgets are properly padded and aligned
- [ ] Layout adapts when window is resized
- [ ] Application launches without errors

## Technical Details

**Implementation Approach:**

1. **Create main window:**
```python
root = tk.Tk()
root.title("Simple Python UI Tool")
root.geometry("800x600")
root.minsize(400, 300)
```

2. **Add title label:**
```python
title_label = tk.Label(root, text="Text Processing Tool",
                       font=("Arial", 16, "bold"))
title_label.pack(pady=10)
```

3. **Add input section:**
```python
input_frame = tk.Frame(root)
input_frame.pack(fill=tk.X, padx=20, pady=10)
tk.Label(input_frame, text="Input:").pack(anchor=tk.W)
input_entry = tk.Entry(input_frame, font=("Arial", 12))
input_entry.pack(fill=tk.X)
```

4. **Add execute button:**
```python
execute_btn = tk.Button(root, text="Execute",
                        font=("Arial", 12),
                        width=20, height=2)
execute_btn.pack(pady=20)
```

5. **Add result display:**
```python
result_frame = tk.Frame(root)
result_frame.pack(fill=tk.BOTH, expand=True, padx=20, pady=10)
tk.Label(result_frame, text="Result:").pack(anchor=tk.W)
result_text = tk.Text(result_frame, font=("Arial", 11),
                      height=10, state='disabled')
result_text.pack(fill=tk.BOTH, expand=True)
```

6. **Add status bar:**
```python
status_bar = tk.Label(root, text="Idle",
                      bd=1, relief=tk.SUNKEN,
                      anchor=tk.W)
status_bar.pack(side=tk.BOTTOM, fill=tk.X)
```

7. **Start main loop:**
```python
root.mainloop()
```

**Layout Strategy:**
- Use `pack()` for vertical stacking
- Use frames to group related widgets
- Set appropriate padding (padx, pady) for spacing
- Use `fill` and `expand` for responsive behavior
