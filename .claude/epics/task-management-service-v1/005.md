# Task: Apply Visual Styling with Qt StyleSheets

## Description

Apply Qt StyleSheets to implement the dark theme visual design specified in the PRD. This includes setting the background color to dark gray (#2b2b2b) and text color to green (#00ff00) for a high-contrast, terminal-style aesthetic.

## Acceptance Criteria

- [ ] Qt StyleSheets applied to the application
- [ ] Background color set to #2b2b2b (dark gray)
- [ ] Text color set to #00ff00 (bright green)
- [ ] Padding applied to labels for visual breathing room
- [ ] High contrast between background and text (WCAG AA compliant)
- [ ] Styling applied in `__init__()` method
- [ ] Visual design matches PRD specifications

## Technical Details

### Implementation Approach

Add stylesheet to `ClockWindow.__init__()` after widget setup:

```python
# Apply styling (at end of __init__, before update_time call)
self.setStyleSheet("""
    QMainWindow {
        background-color: #2b2b2b;
    }
    QLabel {
        color: #00ff00;
        padding: 10px;
    }
""")
```

### Styling Breakdown

1. **QMainWindow**:
   - Background: #2b2b2b (dark gray, RGB: 43, 43, 43)
   - Creates dark terminal-style appearance

2. **QLabel**:
   - Color: #00ff00 (bright green, RGB: 0, 255, 0)
   - Padding: 10px (space around text)
   - Applies to both time_label and date_label

3. **Contrast Ratio**:
   - Green (#00ff00) on dark gray (#2b2b2b)
   - Contrast ratio: ~8.6:1 (exceeds WCAG AA requirement of 4.5:1)
   - Excellent readability for extended viewing

### Key Considerations

1. **Qt StyleSheets Syntax**:
   - CSS-like declarative syntax
   - Applied via `setStyleSheet()` method
   - Cascades to child widgets

2. **Alternative: Individual Widget Styling**:
   - Could use `setStyleSheet()` on each widget separately
   - Using QMainWindow stylesheet is cleaner and more maintainable

3. **Font Styling**:
   - Fonts already set in Task 002 via QFont
   - StyleSheets control colors and layout, not fonts here

4. **Accessibility**:
   - Green-on-dark meets accessibility standards
   - High contrast reduces eye strain
   - Clear, readable from typical viewing distance
