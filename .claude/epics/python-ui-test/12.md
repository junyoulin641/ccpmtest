## Description

**Sorting Implementation:**

- Enable sorting: `table_view.setSortingEnabled(True)`
- Handle in model's `sort()` method
- Use `pandas.DataFrame.sort_values()`
- Emit `layoutAboutToBeChanged` and `layoutChanged` signals

**Filtering/Search:**

- Connect QLineEdit.textChanged signal to filter method
- Filter logic: search across all columns
- Use pandas string matching:
  ```python
  mask = df.astype(str).apply(lambda x: x.str.contains(search_text, case=False)).any(axis=1)
  filtered_df = df[mask]
  ```

**Pagination:**

- Only activate for datasets >10,000 rows
- Page size: 10,000 rows per page
- Slice DataFrame: `df.iloc[start:end]`
- Update page label and button states
- Reset to page 1 on new data load or filter change

**Row Selection:**

- Enable selection: `table_view.setSelectionBehavior(QTableView.SelectRows)`
- Selection mode: `table_view.setSelectionMode(QTableView.ExtendedSelection)`
- Get selected rows for future operations (delete, export selection)

**Table View Configuration:**

```python
self.table_view = QTableView()
self.table_view.setAlternatingRowColors(True)
self.table_view.setSelectionBehavior(QTableView.SelectRows)
self.table_view.setSelectionMode(QTableView.ExtendedSelection)
self.table_view.setSortingEnabled(True)
self.table_view.horizontalHeader().setStretchLastSection(True)
self.table_view.horizontalHeader().setSectionResizeMode(QHeaderView.Interactive)
```

**Data Refresh:**

Connect to DataManager signals:
```python
data_manager = DataManager()
data_manager.dataLoaded.connect(self.refresh_table)
data_manager.dataCleared.connect(self.clear_table)

def refresh_table(self, rows, cols):
    df = self.data_manager.get_data()
    self.model.set_data(df)
    self.update_row_count_label()
```

**Export Current View:**

- Button to export filtered/sorted data
- Use FileHandler.export_file()

**Performance Optimization:**

- Use QTimer for debounced search (wait 300ms after typing stops)
- Lazy loading for pagination
- Cache filtered results
- Use `beginResetModel()` and `endResetModel()` for bulk updates
